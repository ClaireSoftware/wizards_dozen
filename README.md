# Project Title

A simple, cartoon-styled adventure game starring a plucky wizard's apprentice recovering her cake from a greedy wyvern.

## Getting Started


### Prerequisites


```
python >3.0
pygame
```

## Deployment

simply ensure PyGame is installed and run wizards_dozen.py
